    1. 접속 시

    서버에 접속했을 경우, 관리자가 API를 시작하기 전 까진 아무런 변화가 없습니다.
    만약, 관리자가 API를 시작했을 경우 플레이어에게 다음과 같은 변화가 일어납니다.

    2. API의 동작

    관리자가 API를 시작했을 경우, API는 다음과 같이 동작합니다.

    2-1. 후원

    시청자가 치즈를 후원했을 경우 API는 서버에서 후원 알람를 수신한 뒤
    플레이어에게 해당하는 효과를 적용하는 방식으로 동작합니다.
    이 때 후원 금액은 API 리스트에 명시된 것과 "완벽히 일치"해야만 하며,
    스트리머는 이를 시청자에게 자세히 고지 할 의무가 있고,
    일부 항목의 경우 단일성으로 작동하거나, 다른 후원 요소를 비활성화 하는 경우도 있으므로
    반드시 콘텐츠 진행 이전에 시청자에게 해당 사항을 미리 고지해야 합니다.
    (해당 사항을 고지 하지 아니할 경우의 책임은 스트리머 본인에게 있습니다.)

    2-2. 다이아의 사용

    스트리머는 챌린지 달성을 위해 "다이아를 사용"해야 하며,
    이는 다이아를 다른 아이템으로 제작하는 것이 아닌 "인벤토리에서 직접 차감"하는 것입니다.
    해당 기능이 동작하는 방식은 다음과 같습니다.

    인게임에서 OO한 상태로 "Key"를 눌러 다이아를 차감합니다.
    차감 즉시 다이아 할당량의 카운트가 감소하고 차감된 다이아는 복구할 수 없으므로 신중히 선택해야 합니다.

    2-3. 서브퀘스트

    서브 퀘스트는 다이아 할당량과는 별개로 따로 추가되는 퀘스트입니다.
    해당 퀘스트는 "다이아 할당량"을 달성한 경우에도 "유효하며"
    챌린지의 종료는 두 퀘스트를 모두 완료해야 합니다.
    단, 서브 퀘스트는 다이아 할당량을 모두 채웠을 경우 "추가 후원을 통한 할당치 증가"는 불가능합니다.

    2-4. 메인 퀘스트 변경

    시청자가 50만 치즈(받은 김에 끝까지)를 후원했을 경우,
    메인 퀘스트였던 다이아 챌린지 및 서브 퀘스트는 즉시 중단되고 새로운 퀘스트가 주어집니다.
    변경된 퀘스트를 완료할 경우 챌린지 완료처리 되며, 실패시 벌칙을 이행해야 합니다.
